## Create New Angular Project

## 01. Step
```bash
npx create-nx-workspace
```

## 02. Angular Resources
### Generate Module
```bash
nx g @nrwl/angular:module node-v1 --routing --project=store-client
```

## Library

```bash
nx g @nrwl/angular:component ui-card --export --project=app-ui --standalone
```
